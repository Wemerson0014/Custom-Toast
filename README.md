# Custom-Toast

Aplicativo de estudo com objetivo para criação de **Toasts Customizados**. Para cada botão clicado **Sucesso**, **Alerta** ou **Error**, um *Toast* é exibido contendo uma informação e uma cor diferente para o usuário.
